# <span className="text-blue-700">Beautiful</span> state machines

## What is this?

XState Catalogue is a collection of professionally designed state machines you can drop into your projects. Get started by browsing the collection, interacting with the machines, and copying the code.

New here? Our [form input](/machines/form-input) example is a great one to start with.

## Who made this?

My name's Matt. I've been using XState in production for around a year. I also built [xstate-codegen](https://github.com/mattpocock/xstate-codegen). Feel free to consider sponsoring me on [GitHub](https://github.com/sponsors/mattpocock).

## What's the plan?

This is a work in progress. I'm hoping to create a proper library of machines that folks can reference, interact with and learn from.

Imagine 10 times this number of machines, all tagged and searchable...

It's going to be sweet. Follow along on [Twitter](https://twitter.com/mpocock1).
